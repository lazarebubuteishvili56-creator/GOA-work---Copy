💻HTML არის ვებგვერდის სტრუქტურის ენა. ამ კურსში ისწავლი როგორ შექმნა გვერდის ჩონჩხი — ტექსტები, სურათები, ღილაკები, ფორმები და სხვა ელემენტები. კურსის ბოლოს შეძლებ დამოუკიდებლად ააწყო სრული ვებგვერდის სტრუქტურა.

რას შეისწავლი

  • HTML ტეგები და ატრიბუტები
  • Forms და Input ელემენტები
  • რეალური პროექტის აწყობა

🎨CSS გამოიყენება ვებგვერდის გასალამაზებლად. ისწავლი როგორ მიანიჭო ფერები, ფონები, ანიმაციები და შექმნა თანამედროვე დიზაინი.

  • Colors, Fonts, Backgrounds
  • Flexbox და Grid
  • Responsive Design (მობილურზე მორგება)
  • ანიმაციები და ეფექტები

კურსის ბოლოს შენი საიტი იქნება თანამედროვე და პროფესიონალური

JavaScript ვებგვერდს ანიჭებს ინტერაქტიულობას. ღილაკები, მენიუები, კალკულატორი, თამაშები — ყველაფერი JS-ით მუშაობს.

  • Variables და Functions
  • DOM Manipulation
  • Events
  • მარტივი თამაშისა და აპლიკაციის შექმნა

C++ არის ერთ-ერთი ყველაზე ძლიერი და სწრაფი პროგრამირების ენა. გამოიყენება თამაშებში, სისტემურ პროგრამირებაში და ოლიმპიადებში.

  • ცვლადები და მონაცემთა ტიპები
  • ციკლები და პირობითი ოპერატორები
  • ფუნქციები
  • მასივები და ალგორითმები
  • პრობლემების ამოხსნა

კურსი ავითარებს ლოგიკურ აზროვნებას და ალგორითმულ უნარებს
